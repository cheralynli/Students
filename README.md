This program takes a database of students' scores and sorts it.
Input:
[Number of Students]
[Student 1 ID],[Student 1 Name]
[Student 2 ID],[Student 2 Name]
[Number of Grade Records]
[Student ID],[Subject],[Score]
[Student ID],[Subject],[Score]

Output:
Total Students: [the number of all students]
Total Subject: [the nunber of all subjects]
Average Score: [average score of all subject]
Subject Analysis:
[Subject 1]: Avg: [avg1], Max: [max1], Min: [min1], Pass Rate: [pass_rate1]
[Subject 2]: Avg: [avg2], Max: [max2], Min: [min2], Pass Rate: [pass_rate2]
Student Top-3 Rankings:
Student ID Student Name Total Score
[Student 1 ID][Student 1 Name][Total Score 1]
[Student 2 ID][Student 2 Name][Total Score 2]
[Student 3 ID][Student 3 Name][Total Score 3]
